## 核心概念理解

### 神经网络是什么
神经网络 = 输入层 → 隐藏层 → 输出层，每层做加权求和 + 激活，逐层提取特征，最终做出预测。

### 激活函数的作用
没有激活函数，多层 Linear 等价于单层线性变换。ReLU 引入非线性，使网络能学习复杂规律。

### 损失函数的作用
衡量「预测值与真实值的差距」，训练目标是最小化损失。

### 优化器的作用
根据梯度自动更新模型参数，封装了 Day 2 手写的梯度下降逻辑。

- **Adam**: Adaptive learning rate, default choice for most scenarios
- Standard training loop (4 steps):
  1. `optimizer.zero_grad()` — Zero gradients
  2. `model(x)` — Forward pass
  3. `loss = criterion(y_pred, y)` — Compute loss
  4. `loss.backward()` — Backward pass
  5. `optimizer.step()` — Update parameters

## Core Concept Understanding

### What is a Neural Network
Neural network = Input layer → Hidden layers → Output layer. Each layer does weighted sum + activation, extracts features layer by layer, and finally makes predictions.

### The Role of Activation Functions
Without activation functions, stacking multiple Linear layers is equivalent to a single linear transformation. ReLU introduces non-linearity, enabling the network to learn complex patterns.

### The Role of Loss Functions
Measures the gap between predictions and ground truth. The training objective is to minimize the loss.

### The Role of Optimizers
Automatically updates model parameters based on gradients, encapsulating the gradient descent logic manually written on Day 2.
