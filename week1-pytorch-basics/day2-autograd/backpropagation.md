## 为什么需要反向传播？

### 1. 机器学习的核心问题：优化
在机器学习中，我们的根本目标是通过不断调整模型参数来最小化损失函数。反向传播提供了一种高效的方法来计算损失函数相对于每个参数的梯度。

### 2. 传统梯度计算的局限性
- 手动计算梯度：
  - 对于复杂的神经网络，手动求导几乎是不可能的
  - 容易出错，尤其是在多层网络中
  - 计算效率极低

### 3. 反向传播的优势
- **自动化梯度计算**：自动微分技术可以精确地计算每个参数的梯度
- **计算效率高**：通过链式法则高效地传播梯度信息
- **可扩展性强**：适用于各种复杂的神经网络架构
- **减少人工干预**：开发者可以专注于模型设计，而非繁琐的梯度计算

### 4. 反向传播的工作原理
```
输入层 → 中间层 → 输出层 
   ↑           ↑
梯度回传   损失计算
```

- 前向传播：计算预测值
- 反向传播：根据损失函数计算梯度
- 参数更新：使用梯度下降法调整权重

### 5. 实际示例：线性回归
```python
# 前向传播
y_pred = weights * x + bias

# 计算损失
loss = (y_pred - y_true)²

# 反向传播
loss.backward()  # 自动计算梯度

# 更新参数
weights -= learning_rate * weights.grad
bias -= learning_rate * bias.grad
```

### 6. 反向传播的数学本质：链式法则
对于复合函数 z = f(g(x))，其导数计算遵循链式法则：
dz/dx = df/dg * dg/dx

## 实践意义
反向传播使得深度学习成为可能：
- 训练复杂的神经网络
- 自动学习特征表示
- 实现从简单线性模型到深度神经网络的平滑过渡

## 注意事项
- 梯度可能出现vanishing或exploding问题
- 学习率的选择至关重要
- 不同的优化算法会影响收敛速度
